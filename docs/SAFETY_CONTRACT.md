# Safety Contract (Kid Safe First)

## Always disallow
- Nudity/sexual content
- Violence, gore, blood, injury
- Weapons
- Drugs/alcohol
- Hate/harassment
- “Scary horror” (replace with **spooky-cute** only)
- Realistic depictions of real people

## Always allow
- Cute cartoon/sticker styles
- Magic, sparkles, whimsical night scenes (cozy + friendly)
- Food (pancakes, cupcakes), animals, fantasy, space, toys

## “Spooky-cute” definition
Night-glow, stars, friendly ghosts, silly shadows, cozy moonlight.
**No** monsters, terror, gore, chase scenes, screaming.

## Moderation gates
- Moderate composed prompt text before generation
- Moderate generated image before it can be approved
- Block/refuse if flagged; show kid-friendly message:
  - “Let’s try a different idea!”

## Privacy posture (under-13)
- Collect no personal data
- No kid accounts
- No uploads/photos in MVP
- Store minimal metadata (random device UUID)
- Parent controls approvals and exports
